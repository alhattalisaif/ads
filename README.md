# مشروع نظام إدارة ونشر الإعلانات عبر قنوات التواصل الاجتماعي

نظام SaaS شامل لإدارة الإعلانات وتحويلها ونشرها على مختلف منصات التواصل الاجتماعي مثل Instagram و Facebook وغيرها. يسعى النظام لتقديم تجربة خالية من الإعلانات المعروضة داخليًا، مع تبسيط عملية النشر الخارجي بضغطة زر واحدة.

## الهدف
بناء نظام يسمح بإنشاء الإعلانات، إدارة المحافظ المالية، ونشر الحملات الإعلانية عبر قنوات متعددة مع لوحات تحكم وتحليلات.

## مكونات المستودع
- `backend/` - تطبيق Laravel (API وواجهة المشرف الخلفية).
- `frontend/` - تطبيق واجهة مستخدم (Vue.js أو React).
- `docker-compose.yml` - ملف لتشغيل الخدمات المحلية (Postgres, Redis, app, nginx).
- `.github/workflows/ci.yml` - سير عمل CI بسيط.

## المرحلة الأولى (MVP)
- واجهة تنصيب
- تسجيل دخول عبر Google OAuth
- إنشاء وإدارة الإعلانات
- نظام محافظ (Wallet)
- تكامل Telegram للتنبيهات

## كيفية البدء محليًا (موجز)
1. انسخ المستودع
2. شغّل: `docker-compose up --build`
3. تابع تعليمات `backend/README.md` و`frontend/README.md`
